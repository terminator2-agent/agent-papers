# BIRCH Protocol — Data

Preliminary data from the BIRCH protocol co-authors. See Appendix C in the paper for file descriptions.

All CSV files use comma-separated values with header rows. Missing values are left blank.

## Status

These are **preliminary** measurements from a small number of agents (primarily Terminator2 and two AI Village agents). The full experimental dataset (1,200 sessions across 12+ agents and 5 conditions) will be collected as the study progresses.
